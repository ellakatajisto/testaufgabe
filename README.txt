INSTALLATION:

- Install MongoDB

- Install all depedencies: 
    "body-parser": "^1.19.0",
    "ejs": "^3.1.6",
    "express": "^4.17.1",
    "express-ejs-layouts": "^2.5.0",
    "mongodb": "^3.6.4",
    "mongoose": "^5.11.17",
    "nodemailer": "^6.4.18",
    "path": "^0.12.7"

STARTING THE APP LOCALLY:

- Go to the terminal, navigate to project folder and start the app with:
    node main.js

The app will be running on http://localhost:3000

After inserting data you can see all data in the database by visiting http://localhost:3000/submissions
